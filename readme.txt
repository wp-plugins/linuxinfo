== Description ==

LinuxInfo is a port of Linfo (under GPL) and is adapted for working easily, quickly and properly with WordPress.
<i>Original Idea / Concept: https://wordpress.org/support/topic/minimalist-server-load-stats-plugin-in-black-bar</i>

An brief introduction of what you would display:

<strong>General (Core)</strong>

<ul>
<li>Distibution</li>
<li>Kernel Version</li>
<li>Accessed IP</li>
<li>Uptime</li>
<li>Hostname</li>
<li>Architecture</li>
</ul>

<strong>Load, Network</strong>

<ul>
<li>Processes</li>
<li>Threads & Active Users</li>
<li>Memory</li>
<li>Network Devices (and Load)</li>
<li>Filesystem Mounts (and Load)</li>
<li></li>
</ul>

<strong>And More!</strong>
